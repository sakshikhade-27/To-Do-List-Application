# To-Do List Manager (Python)

This project is a simple To-Do List Manager developed using Python and Jupyter Notebook.
It helps users manage daily tasks efficiently.

## Features
- Add new tasks
- View all tasks
- Update task status (Completed / Pending)
- Delete tasks
- Simple menu-driven interface

- ## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Open Jupyter Notebook
2. Open `todo.ipynb`
3. Run all cells
4. Use the menu options to manage tasks
