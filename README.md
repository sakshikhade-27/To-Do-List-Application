# To-Do List Manager (Python)

This project is a simple To-Do List Manager developed using Python and Jupyter Notebook.
It helps users manage daily tasks efficiently.

# Problem Statement
Create a To-Do List Manager application in Python that helps users manage daily tasks 
effectively. The system should allow users to add new tasks, view existing tasks, update task 
status (completed/pending), and delete tasks. This project focuses on building an interactive, 
user-friendly program while strengthening problem-solving and data manipulation skills.

## Features
- Add new tasks
- View all tasks
- Update task status (Completed / Pending)
- Delete tasks
- Simple menu-driven interface

- ## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Open Jupyter Notebook
2. Open `todo.ipynb`
3. Run all cells
4. Use the menu options to manage tasks
